# sdl_game

sdl game demo

### Dependencies

- [SDL2 release-2.26.2](https://github.com/libsdl-org/SDL/tree/SDL2)
- [logc](https://github.com/realjf/logc.git)

### Build

**`Linux`**

```sh
git clone --recursive https://github.com/realjf/sdl_game.git
cd sdl_game
make build
make run
```

**`Windows`**

```sh
git clone --recursive https://github.com/realjf/sdl_game.git
cd sdl_game
make build PLAT=WINDOWS
make run PLAT=WINDOWS

```
